== Description ==

Flip your Product Category list Images with the Gallery Images


After install the plugin the flipping will be activated

= Need Support ? =

*Visit here for support https://woocommerce-img-flipper.myridia.com



Select which categories of product want to be flip For example T-shirt,Mobiles to show front and back side view.

== Installation ==

1. Upload `woocommerce-img-flipper` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
